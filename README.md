# Semana15 - Actividad MQTT Android

Este proyecto Android implementa una app que se conecta a HiveMQ Cloud usando MQTT v3 con TLS y autenticación. Permite publicar y recibir mensajes en tiempo real desde un tópico específico.

## Características
- Cliente MQTT usando HiveMQ Client
- Conexión segura con usuario y contraseña
- Publicación y suscripción al tópico `prueba1`
- Interfaz simple con `EditText`, `Button` y `TextView`
- Manejo robusto de errores y validación de conexión

## Broker usado
- HiveMQ Cloud
- Host: `da53951e1d5143b891928f79d75faff0.s1.eu.hivemq.cloud`
- Puerto: `8883`
- Usuario: `prueba1`

## Requisitos
- Android Studio
- Java 8+
- Internet activo para conexión MQTT
